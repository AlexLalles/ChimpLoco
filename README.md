# ChimpLoco

**Hand-based VR locomotion for Unity 6 (tested in 6.3.0f1).**  

This movement system is inspired by Gorilla Tag's locomotion, but fully rewritten and independently implemented.

## Features
- VR hand-based movement
- Highly customizable
- Haptics support (requires Easy Inputs)
- Tap sounds
- And a few other things.

## Dependencies
- Unity 6 (tested in 6.3.0f1)
- Easy Inputs
- XR Interaction Toolkit
- OpenXR
- URP (recommended)
